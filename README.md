# Jeu de Devinettes

Ce projet est un jeu où il faut deviner un nombre.
L'ordinateur choisit un chiffre entre 1 et 100.

## Règles du jeu
- L'utilisateur a 10 essais maximum.
- Le jeu indique si c'est "plus" ou "moins".
- On gagne si on trouve le juste prix !

## Structure du code
- `Game.java` : contient toute la logique du jeu.
- `Scanner` : utilisé pour lire les entrées.
- `Random` : utilisé pour générer le nombre secret.

## Installation
1. Cloner le dépôt GitHub.
2. Importer sous IntelliJ avec le JDK 21.
3. Lancer la classe principale.
